# Gizlilik Politikası / Privacy Policy

---

## Türkçe

### Veri Sorumlusu

PulsePoll'un veri sorumlusu **Haluk Eren Özlü**'dür.
İletişim için GitHub profili üzerinden ulaşabilirsiniz.

### Hangi Veriler Toplanır?

PulsePoll iki tür kullanıcıya hitap eder:

**Kayıtlı kullanıcılar:**

- E-posta adresi
- Şifre (hashlenmiş olarak saklanır; düz metin hiçbir zaman tutulmaz)
- Anket oluşturma geçmişi
  **Misafir (kayıtsız) kullanıcılar:**
- Oy verme işlemi için kısa ömürlü, anonim bir misafir kimliği (`guest_id`) bir çerez (cookie) aracılığıyla tarayıcıda saklanır.
- Bu kimlik kişisel bilgi içermez ve anket süresi dolduğunda geçerliliğini yitirir.

### Veriler Nasıl Kullanılır?

Toplanan veriler yalnızca şu amaçlarla kullanılır:

- Hesap oluşturma ve oturum yönetimi
- Tekrarlı oy ve kötüye kullanımı önleme
- Anket kurallarının uygulanması
  Veriler üçüncü taraflarla paylaşılmaz, reklam amacıyla kullanılmaz.

### Oy Gizliliği

Bireysel oy kayıtları saklanmaz. Yalnızca seçenek bazında toplam oy sayıları tutulur. Bu tasarım, kimin hangi seçeneğe oy verdiğinin tespit edilememesini sağlar.

### Altyapı

Proje şu an geliştirme aşamasındadır. Canlı ortama geçildiğinde kullanılacak altyapı sağlayıcıları bu belgede güncellenecektir.

### Değişiklikler

Bu politika proje geliştikçe güncellenebilir. Önemli değişiklikler GitHub üzerinden duyurulur.

## English

### Data Controller

The data controller for PulsePoll is **Haluk Eren Özlü**.
You can reach out via the GitHub profile linked on this repository.

### What Data Is Collected?

PulsePoll serves two types of users:

**Registered users:**

- Email address
- Password (stored as a hash; plain text is never retained)
- Survey creation history
  **Guest (unregistered) users:**
- A short-lived, anonymous guest identifier (`guest_id`) is stored in the browser via a cookie for the purpose of voting.
- This identifier contains no personal information and expires when the survey ends.

### How Is Data Used?

Collected data is used solely for:

- Account creation and session management
- Preventing duplicate votes and abuse
- Enforcing survey rules
  Data is not shared with third parties and is not used for advertising.

### Vote Privacy

Individual vote records are not stored. Only aggregate vote counts per option are retained. This design ensures that it is not possible to determine which option a specific person voted for.

### Infrastructure

The project is currently in development. When deployed to a live environment, the third-party infrastructure providers used will be listed and updated in this document.

### Changes

This policy may be updated as the project evolves. Significant changes will be announced via GitHub.
